설치부터 사용법까지 — 6단계면 끝

아직 Chrome 웹 스토어에 정식 등록 전이라, 압축해제된 확장 프로그램으로 직접 설치합니다.

1

파일을 내려받고 압축을 풉니다

아래 버튼으로 SMARTSHOT 압축 파일을 받아 원하는 폴더에 압축을 풀어주세요. 다운로드 폴더에 바로 풀지 마시고, 바탕화면이나 별도 폴더에 풀어주세요 — 캡쳐 이미지가 저장되는 폴더와 헷갈리지 않습니다.

⬇ SMARTSHOT_확장프로그램.zip 다운로드

⚠️ 처음 받을 때 크롬이 "이 파일은 일반적으로 안전하지 않을 수 있습니다" 같은 확인 메시지를 띄울 수 있어요. 위험한 게 아니라 잘 알려지지 않은 사이트의 압축 파일이라 한 번 확인하는 정상적인 절차입니다. "유지"(계속) 버튼을 누르면 다운로드가 이어집니다.
2

Chrome 주소창에 chrome://extensions 를 입력합니다

주소창을 클릭해서 위 주소를 직접 입력하거나 복사해서 붙여넣기(Ctrl+V) 해도 됩니다 (보안상 이 글의 링크를 클릭해서는 열리지 않아요). 이동한 뒤 우측 상단의 개발자 모드를 켜주세요.

💡 위 chrome://extensions 글자를 드래그해서 복사(Ctrl+C)한 다음, 주소창에 붙여넣기(Ctrl+V) 하시면 편해요.

개발자 모드
3

좌측 상단의 압축해제된 확장 프로그램을 로드 버튼을 누릅니다

화면 좌측 상단에 나타난 버튼을 누르고, 1단계에서 압축을 푼 SMARTSHOT_확장프로그램 폴더를 선택하면 설치가 끝납니다.

확장 프로그램
압축해제된 확장 프로그램을 로드 확장 프로그램 압축 업데이트
4

이제 페이지를 열면 우측에 작은 버튼이 떠 있어요

어떤 웹사이트든 우측 가장자리에 ✂️ 영역선택 버튼이 자동으로 뜹니다. 더 많은 기능이 필요하면 ▾를 눌러 화면캡쳐 · 전체캡쳐 · OCR추출 버튼을 펼치세요.

5

원하는 캡쳐 방식을 고르고, 결과 창에서 다듬습니다

영역선택은 드래그한 부분만, 화면캡쳐는 지금 보이는 화면 전체, 전체캡쳐는 스크롤이 필요한 긴 페이지 전체를 담습니다. 결과 창에서는 사각형 · 원형 · 화살표 · 선 · 텍스트 · 모자이크로 강조하거나 가릴 수 있고, 🔤 OCR 텍스트 추출로 글자도 바로 뽑아냅니다.

6

⚙ 설정에서 내 방식대로 바꿔보세요

버튼 크기(대/중/소), 저장 폴더, 파일 형식(PNG/JPG), 결과 창 자동 열기 여부, 캡쳐 시 자동 저장 여부까지 모두 설정에서 켜고 끌 수 있습니다.

✂️

영역선택 캡쳐

드래그한 영역만 잘라서 캡쳐하고, 동시에 클립보드에 자동 복사됩니다.

🖥

화면캡쳐

지금 화면에 보이는 부분만 한 번에 캡쳐합니다.

🖼

전체페이지 캡쳐

스크롤이 필요한 긴 페이지도 자동으로 스크롤하며 이어 붙여 한 장으로 만듭니다.

🔤

OCR 텍스트 추출

한국어 · 영어 텍스트를 이미지에서 바로 인식해 복사할 수 있는 텍스트로 뽑아줍니다.

⬜

주석 도구 6종

사각형 · 원형 · 화살표 · 직선 · 텍스트 · 모자이크. 색상 · 두께 · 굵기 · 배경색까지 따로 조절합니다.

📋

클립보드 자동 복사

영역선택만 해도 바로 붙여넣기(Ctrl+V) 할 수 있게 클립보드에 복사됩니다.

💾

저장 위치 · 형식 선택

다운로드 폴더 바로 저장 또는 하위 폴더 저장, PNG/JPG 형식까지 설정에서 고를 수 있습니다.

🪟

결과 창 나란히 보기

캡쳐 결과 창이 화면 우측에 뜨고, 작업 중인 창은 자동으로 좌측에 맞춰져 한눈에 비교됩니다.

🗂

캡쳐 히스토리

방금까지 캡쳐한 화면들이 썸네일로 쌓이고, 클릭하면 언제든 다시 불러와 편집할 수 있습니다.
